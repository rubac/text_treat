# Text as Data and Causal Inference in Sociology
This repository contains the code for the paper "Text as Data and Causal Inference in Sociology" by Nicole Schwitter, Ruben Bach, and Christopher Klamm. The paper will be published at https://doi.org/10.1007/s11577-026-01071-y
## Survey Data (text as outcome) 
The text-as-outcome application uses data from wave 76 of the German Internet Panel (GIP). The text data used are only available through the on-site data access at the University of Mannheim. Labelled data can be made available upon request to researchers which have access to the GIP data. See for information on access: (https://www.uni-mannheim.de/en/gip/using-gip-data/)
## Survey Data (text as treatment)
The survey data can be requested via the Harvard Dataverse (https://doi.org/10.7910/DVN/SEV0SZ)
